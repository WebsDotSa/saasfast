تحقق
Migrations 014-016 — تأكد من تطبيقها في Supabase
Team Management + Announcements + Referrals موجودة في الكود لكن جداولها في Supabase يحتاج تأكيد. ستظهر أخطاء عند استخدامها إذا لم تُطبَّق.
-- تحقق في Supabase SQL Editor: SELECT table_name FROM information_schema.tables WHERE table_name IN ('team_invitations','announcements','referrals','referral_rewards');
إصلاح
TenantThemeProvider — loading screen بلا timeout
إذا فشل أو أبطأ /api/settings، المستخدم يرى شاشة فارغة للأبد. الحل: أضف 3s timeout.
// في tenant-theme-provider.tsx const timeout = setTimeout(() => setLoading(false), 3000); fetchBranding().finally(() => { clearTimeout(timeout); setLoading(false); });
ملاحظة
user_profiles table — مطلوبة للـ Admin Panel
Admin layout + Users page + Tenant detail تقرأ من user_profiles. إذا لم تكن موجودة في DB سيحدث خطأ. تأكد من وجودها أو أضف migration لها.

⚡ المستوى 1 — AI Utilities (أسبوع واحد)
وظائف AI مساعدة جاهزة تُستخدم في أي module. تثبّت مرة واحدة وتُستدعى في أي مكان.
@ai-sdk/anthropic lib/ai/index.ts generateText() + streamText() Arabic-first
## الخطوة 1: تثبيت pnpm add ai @ai-sdk/anthropic @ai-sdk/openai ## الخطوة 2: lib/ai/index.ts — AI Utilities import { generateText, streamText } from 'ai' import { anthropic } from '@ai-sdk/anthropic' export async function generateArabicContent(prompt: string) { const { text } = await generateText({ model: anthropic('claude-sonnet-4-5'), system: 'أنت مساعد متخصص للسوق السعودي. رد بالعربية دائماً.', prompt, }) return text } export async function analyzeArabicText(text: string) { ... } export async function translateToArabic(text: string) { ... } export async function generateSEOContent(product: any) { ... }
🧠 المستوى 2 — AI per Module (أسبوعان)
كل Module يحصل على مساعد AI مخصص يفهم سياق المجال. يُضاف كـ Server Action في كل صفحة.
E-commerce AI Page Builder AI HR AI CRM AI Accounting AI

// src/app/api/ai/agent/route.ts — AI Agent with Tools import { streamText, tool } from 'ai' export async function POST(req: Request) { const { messages, tenantId } = await req.json() const result = await streamText({ model: anthropic('claude-sonnet-4-5'), system: buildTenantSystemPrompt(tenantId), // سياق المنشأة messages, tools: { getProducts: tool({ description: 'احصل على منتجات المنشأة', parameters: z.object({ limit: z.number() }), execute: async ({ limit }) => getProductsForTenant(tenantId, limit) }), createOrder: tool({ ... }), checkInventory: tool({ ... }), processPayment: tool({ ... }) } }) return result.toDataStreamResponse() }
